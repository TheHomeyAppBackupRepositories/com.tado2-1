Tid til smartere opvarmning. Udskift din gamle termostat med tado° Smart Thermostat og styr din opvarmning hvor som helst fra.
Forbind tado° termostaten via Homey med dine andre enheder i hjemmet og oplev et nyt niveau af komfort, mens du sparer energi og omkostninger.

tado° Radiator Termostater kan ikke tilsluttes Homey uden tado° bridge.

Indstil dine præferencer for manuel kontrol med Homey via de avancerede indstillinger på din enhed.

Åben vindue detektering kan desværre kun kontrolleres hvert 15. minut.
